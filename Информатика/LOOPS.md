# Циклы в программировании

## 1. Что это такое?

**Цикл** — это управляющая конструкция, которая позволяет выполнить один и тот же блок кода многократно (не копируя его вручную).
## 2. Для чего используется?

1. Обработка больших данных (списки, строки, файлы, базы данных).
2. Избежание дублирования кода (DRY-принцип: Don't Repeat Yourself).
3. Выполнение действий до наступления условия (ожидание ввода, опрос датчика).
4. Автоматизация повторяющихся задач (отправка 1000 писем, рендер кадров игры).
## 3. Код (Python)

По циклу for 
```python
# Вывести числа от 0 до 4
for i in range(5):  # range(stop) -> 0, 1, 2, 3, 4
    print(f"Итерация номер {i}")

# range(2, 10, 2) -> от 2 до 9 с шагом 2
for четное in range(2, 10, 2):
    print(четное)  # Вывод: 2, 4, 6, 8
```

По циклу While 

```python
# Угадай число (игрок вводит пароль)
password = ""
while password != "сезам":
    password = input("Введите пароль: ")
    if password != "сезам":
        print("Неверно, пробуй еще...")

print("Дверь открыта!")

# Осторожно: бесконечный цикл (while True: print('Вечно'))
# Всегда должен быть способ выйти (break или изменение условия)
```
## 4. Когда что использовать?

for — для перебора коллекции или фиксированного диапазона (95% задач).
while — когда условие зависит от внешних факторов (ввод пользователя, точность вычислений, ожидание сигнала).
