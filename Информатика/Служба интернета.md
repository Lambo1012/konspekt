 Служба интернета - это протоколы и системы, которые обеспечивают обмен данными между устройствами. Современные службы работают по модели TCP/ IP. 
### Ключевые принципы

1. Клиент серверная модель, где клиент делает запрос,  а сервер на него отвечает. 
2. Порты - каждая служба использует собственный порт для обмена запросами. 
3. Безопасность - почти у каждой службы есть защищённая версия.
