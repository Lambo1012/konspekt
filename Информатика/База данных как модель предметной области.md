
База данных (БД) - это совокупность данных организованных по определенным правилам отражающие состояние объектов и их отношений в  некоторой предметной области предназначенная для хранение постоянной памяти и её применения  . 
БД можно рассматривать как информационную модели предметной области.
Предметная область это целевая тема конкретного проекта. Свою очередь доменная или концептуальная модель призвано выразить значение терминов и понятий а также для  установление верных взаимосвязей между различными понятиями.

### 3 основных типа БД  
1. Реляционная БД - называют табличными, потому что в них можно представить в виде таблиц 
2. Сетевые БД - может быть несколько различных связей, в том числе несколько общих предков. Особенности сетевой БД в том то в ней запоминаются все связи для каждой из них. Иначе говоря поиск в такой БД индексируются.
3. Иерархические БД - модель организующая данные в древовидную структуру ,где каждый дочерний элемент подчиняется только одному родительскому.
