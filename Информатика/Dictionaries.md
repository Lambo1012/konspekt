## Что такое словарь простыми словами?

Словарь — это как коробка с наклейками. На каждой наклейке написано название (ключ), а внутри лежит какая-то вещь (значение). Ты можешь быстро найти нужную вещь по наклейке, не перебирая всю коробку.
### В реальной жизни:
- Телефонная книга: имя человека (ключ) → номер телефона (значение)
- Шкафчик в спортзале: номер ключа (ключ) → сумка внутри (значение)
### В программировании:
- Словарь связывает уникальный ключ с любым значением
- Поиск по ключу происходит мгновенно (мгновенно — даже в огромном словаре)

## Где пригодится словарь в коде?

1. Хранить данные об объекте (пользователь, товар, заказ)
2. Быстро искать что-то по уникальному коду (ID, имя, номер)
3. Считать количество чего-либо (сколько раз встречается каждое слово)
4. Настраивать программу (параметры и их значения)
5. Создавать кэш (запоминать результаты, чтобы не пересчитывать)

```python
# Пустой словарь
коробка = {}

# Сразу с данными
пользователь = {
    "имя": "Анна",
    "возраст": 25,
    "город": "Москва"
}

print(пользователь["имя"])  # Выведет: Анна
```
