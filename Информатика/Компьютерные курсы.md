

Компьютерная сеть - это два или более компьютеров, соединенных между собой  для обмена информацией и совместного использования ресурсов. 


Узел сети : Любое устройство в сети. 

Сервер: Устройство, представляющее ресурсы или слуги. 

Клиент: Устройство, которое использует эти услуги

Сетевой кабель : Беспроводная связь 


### Локальная сеть 

Локальная сеть - это  сеть, ограниченная небольшими пространствами офис.

Основные цели - быстрый обмен данными внутри сети. 

Главное устройство : Коммутатор - Интеллектуальный разветвитель, который 
направляет данные точно целевому устройство в локальную сеть. 

### Глобальная сеть 


Глобальная сеть есть интернет. Она объединяет  миллионы локальных мира, всему миру. 

Если LAN - это  улица в вашем городе, то Интернет - это все дороги мира, связывающие города и страны. 


##### Как это работает 

Вша локальная сеть подключается к провайдеру , а провайдер соединяются между собой. 

Главное устройство для выхода в Интернет: Маршрутизатор . Его задача - проложить путь для данных из вашей локальной сети в глобальную и обратно. Он стоит на границе между LAN и Интернетом. 


### LAN и Интернет  


| Характеристика      | Локальная        | Глобальная сеть      |
| ------------------- | ---------------- | -------------------- |
| Масштаб             | Комната, здание  | Вся планета          |
| Скорость            | Высокая          | Средняя              |
| Управление          | Одна организация | Никем не укравляется |
| Основное устройство | Коммутатор       | Маршрутизатор        |
|                     |                  |                      |
