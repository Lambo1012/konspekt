В Python **исключения (exceptions)** — это ошибки, которые возникают во время выполнения программы. Если их не обрабатывать, программа аварийно завершается.

### 1. Базовый шаблон (try - except)

В try пишем опасный код, в except — что делать при ошибке.
### 2. Полный цикл (else и finally)

- else — сработает, если в try не было ошибок.
- finally — сработает всегда (подходит для закрытия файлов/соединений).
### 3. Перехват текста ошибки (as e)

Позволяет узнать, что именно пошло не так.

```python
try:
    print([1, 2][5])  # Индекса 5 нет
except IndexError as e:
    print(f"Ошибка: {e}")  # Выведет: list index out of range
```
