наборы это как списки только там все элементы уникальные и без повторений и еще там нет порядка то есть нельзя взять элемент по индексу. используют наборы чтобы убирать дубликаты из данных например если в списке много одинаковых значений превращаешь его в набор и все повторы исчезают. еще в наборах очень быстрый поиск намного быстрее чем в списках поэтому если тебе нужно часто проверять есть ли элемент то бери набор. еще с наборами можно делать математические штуки как в школе объединение когда собираешь все элементы из двух наборов пересечение когда берешь только общие элементы разность когда берешь из первого чего нет во втором и симметричная разность когда берешь элементы которые есть только в одном из наборов. еще наборы удобны когда нужно хранить уникальные вещи типа список друзей без повторов или айдишники которые не должны повторяться. главное отличие от списка в списке может быть много одинаковых и есть порядок а в наборе все уникальны и порядка нет. короче наборы юзай когда надо убрать дубликаты быстро найти элемент или сделать операции с множествами а списки когда важен порядок или нужно менять данные.

# 1. Удаление дубликатов из списка

```python
список = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]
набор = set(список)
уникальный_список = list(набор)
print(уникальный_список)  # [1, 2, 3, 4]
```

# 2. Проверка наличия элемента

```python
набор = {"яблоко", "банан", "груша"}

if "банан" in набор:
    print("банан есть в наборе")  # быстро работает

if "апельсин" not in набор:
    print("апельсина нет")
```
