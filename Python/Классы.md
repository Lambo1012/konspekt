### Введение

[](https://github.com/nakkapohv/Notes-RPO/blob/main/%D0%92%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5%20%D0%B2%20Python/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D1%8B.md#%D0%B2%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5)

Класс - это пользовательский тип данных, конструкция ООП (Объективно-Ориентированного Программирования), которая описывает тот или иной объект, его свойства и способы взаимодействия с ним.

Класс имеет поля - переменные, которые будут принадлежать к его экземпляру, а так же методы функции, которые также можно будет использовать для его экземпляра.

Простыми словами класс - это шаблон для реализации какого-то объекта. Например, мы знаем как должна выглядеть кружка, какие характеристики и действия она имеет. При помощи класса мы описываем "абстрактную маму-кружку", а экземпляры будут являться настоящими (конкретными) кружками с характеристиками и действиям "мамы-кружки".

Класс решает несколько важных задач:

- Дублирование кода
- Унификация объектов (т.е. одинаковые по задумке объекты действительно будут иметь одинаковые характеристики и функционал)

#### Классы в Python

[](https://github.com/nakkapohv/Notes-RPO/blob/main/%D0%92%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5%20%D0%B2%20Python/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D1%8B.md#%D0%BA%D0%BB%D0%B0%D1%81%D1%81%D1%8B-%D0%B2-python)

Объявляются при помощи ключевого слова **class** и указания имени класса, затем ставится двоеточие и со следующей строки вписывается тело класса.

```python
class Python:
	pass
```

Чтобы объявить поля класса используется ключевое слово self, после которого, через точку указывается имя поля. Поле тоже самое что и переменная, но внутри класса. Одна через self, можно обращаться только из тела метода. В параметрах метода так же обязательно должен быть указан self.

```python
class Python:
	def test(self):
	self.msg = "Hello"
```

Для инициализации полей при создании экземпляра класса используется функция конструктор. Она обязательно принимает параметр self. Эта функция будет автоматически вызвана при создании экземпляра класса. Если она не объявлена, будет использован конструктор по умолчанию (пустой).

```python
class Python:
	#Коструктор
	def __init__(self, text):
	self.msg = "Hello"
	
python_exemple = Python("Hello")
```

Cоздание экземпляра класса (объект класса, объект) происходит вне тела класса. При помощи указания имени класса и круглых скобок после имени. Если в конструкторе кроме self указаны другие параметры, то при создании объекта в скобках мы также укажем атрибуты.
