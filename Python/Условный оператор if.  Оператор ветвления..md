#условный #конспект #els #elif #python 
В python, как и  в других ЯП, существует условный оператор или оператор ветвления. Он позволяет управлять потоком выполнения программы при помощи реализации механизма динамического ветвления на основе оценки булевых выражений . Т.е., if и elifопираеются на истинность или ложность логического выражения и выполняет соответствующий блок кода . 

### Синтаксис и структур 

```python 
if <условие>
	<блок инструкций_1>  
	<блок инструкций_2>  
	<блок инструкций_3>  
```
Если условия выполняется, т.е. истинно, то тогда выполняется и код внутри блока if. Блок кода внутри if всегда начинается с отступа, который равен 4 пробелам или одному табу от родительского if. 

Связка if-elif-els "если, иначе если, иначе"



 ```python
 if <условие_1>:
	 <блок_инструкций_1>
	  
elif <условие_2>:
	 <блок_инструкций_2> 
	 
else:  
	 <блок_инструкций_3> 
 ```
