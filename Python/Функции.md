
Функция — это готовый кусок кода, у которого есть имя. Его можно вызывать сколько угодно раз в разных частях программы, чтобы не писать одно и то же.
Функция нужна, чтобы не писать один и тот же код дважды.

Зачем нужны функции? (кратко)

1. Чтобы не повторяться.  
   Написал код один раз внутри функции — вызывай сколько хочешь. Меньше строк — меньше ошибок.
2. Чтобы код было легче читать.  
   Вместо тысячи строк видно: загрузить, обработать, вывести. Сразу понятно, что делает программа.

3. Чтобы легко искать ошибки.  
   Ошибка внутри одной функции — проверил 5 строчек, а не весь код.

4. Чтобы использовать код в других проектах.  
   Написал полезную функцию — скопировал в новый проект. Экономлю время.

5. Чтобы переменные не мешали.  
   Внутри функции свои переменные, снаружи — свои. Не конфликтуют.

6. Чтобы упрощать сложные задачи.  
   Разбиваю большую задачу на маленькие шаги-функции. Сложное становится простым.

7. Чтобы легко вносить изменения.  
   Поменял логику внутри функции — всё заработало по-новому. Не ищу по всему коду.
