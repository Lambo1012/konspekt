
словари это как телефонная книжка где каждому имени ключ соответствует номер телефона значение. они нужны чтобы быстро находить данные по ключу потому что ищут очень быстро намного быстрее чем списки. у каждого ключа имя уникальное и если добавить существующий ключ то он перезапишет старое значение. ключами могут быть только неизменяемые типы типа строки числа или кортежи а списки нельзя. значениями может быть что угодно даже списки или другие словари. словари удобны когда хранишь данные о чем-то например о человеке имя возраст город и потом быстро получить возраст. еще используют для настроек программы или чтобы считать сколько раз слова встречаются в тексте. чтобы взять значение пишешь словарь чтобы добавить словарь значение. короче словари для быстрого поиска по ключу а списки для порядка и индексов.


# 1. Создание словаря и получение значения по ключу

```python
# Создаем словарь с данными о человеке
человек = {
    "имя": "Иван",
    "возраст": 25,
    "город": "Москва"
}

# Получаем значение по ключу
print(человек["имя"])        # Иван
print(человек.get("город"))  # Москва (безопасный способ)

# Если ключа нет - ошибки не будет
print(человек.get("работа", "не указано"))  # не указано
```

# 2. Перебор словаря (цикл по ключам и значениям)

```python 
оценки = {
    "Маша": 5,
    "Петя": 4,
    "Коля": 3
}

# Перебираем ключи
for имя in оценки:
    print(имя)  # Маша Петя Коля

# Перебираем ключи и значения
for имя, оценка in оценки.items():
    print(имя, "получил", оценка)
```
