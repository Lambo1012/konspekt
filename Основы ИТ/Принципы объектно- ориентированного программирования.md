
### Инкапсуляция 
**Инкапсуляция** — это один из четырёх основных принципов объектно-ориентированного программирования (ООП), наряду с наследованием, полиморфизмом и абстракцией.

```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner  # публичное поле
        self.__balance = balance  # приватное поле (два подчеркивания)
        self._pin = "1234"  # защищенное поле (одно подчеркивание)
    
    # Публичные методы (интерфейс)
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
    
    def withdraw(self, amount):
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            return True
        return False
    
    # Геттер для приватного поля
    def get_balance(self):
        return self.__balance
```

### Наследование 

**Наследование** — это механизм в объектно-ориентированном программировании, который позволяет создавать новые классы на основе существующих, заимствуя их свойства и поведение.
```python
class Vehicle:
    def move(self):
        return "Двигается"

class Car(Vehicle):
    pass  # Получает move() от родителя

car = Car()
print(car.move())  # "Двигается"
```

### Полиморфизм 

**Полиморфизм** (от греч. "много форм") — это способность объектов с одинаковым интерфейсом (именем метода) иметь разную реализацию

```python
class CreditCard:
    def pay(self, amount):
        return f"Оплачено картой: {amount}"

class PayPal:
    def pay(self, amount):
        return f"Оплачено PayPal: {amount}"

# Кассиру все равно, как платят
payment_methods = [CreditCard(), PayPal()]
for method in payment_methods:
    print(method.pay(1000))
```
