WinGet — это официальный менеджер пакетов от Microsoft, встроенный в Windows 10 и 11, который позволяет управлять программами через командную строку: чтобы установить приложение, достаточно знать его ID (например, winget install Google.Chrome ), для поиска используется команда winget search название , для просмотра установленных программ — winget list , для обновления всех сразу — winget upgrade --all а для удаления — winget uninstall имя . Это удобный инструмент для быстрой настройки новых компьютеров и автоматизации установки софта, поддерживающий все популярные форматы установщиков (EXE, MSI, MSIX), но требующий прав администратора и точного знания имен пакетов.
