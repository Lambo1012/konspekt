
1. Просмотр списка пользователей
cmd
net user
2. Изменение пароля пользователя
cmd
net user ИмяПользователя НовыйПароль
_Пример:_ `net user Ivan Qwerty123`

3. Просмотр данных пользователя 
cmd
net user ИмяПользователя
_Пример:_ `net user Ivan`
4. Просмотр локальных групп
cmd
net localgroup
5. Назначение пользователя в локальную группу
cmd
net localgroup ИмяГруппы ИмяПользователя /add
_Пример:_ `net localgroup Администраторы Ivan /add`
6. Запрет на изменение пароля
cmd
net user ИмяПользователя /passwordchg:no
_Разрешить:_ `/passwordchg:yes`
7. Удаление пользователя
cmd
net user ИмяПользователя /delete
_Пример:_ `net user Ivan /delete`
ВАЖНО:
- Команды 2, 5-7 требуют **прав администратора**
    
- Для отмены действий используйте:
    
    - Удаление из группы: `/delete` вместо `/add`
        
    - Разблокировка пароля: `/passwordchg:yes`
